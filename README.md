# TripPlannerLite (iOS)

A simple SwiftUI trip planner that supports creating and deleting trips with on-device persistence using SwiftData.

## Features
- Add trips with a name
- View trips in a list
- Swipe to delete
- Data persists across app restarts (SwiftData)



## Tech
- SwiftUI
- SwiftData
- iOS Simulator

## How to Run
1. Clone the repo
2. Open `TripPlannerLite.xcodeproj` in Xcode
3. Select an iPhone simulator and press Run ▶️
