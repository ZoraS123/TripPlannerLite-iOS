# TripPlannerLite (iOS)

A lightweight iOS trip planner built with **SwiftUI** and **SwiftData** that enables users to create, view, and delete trips with persistent on-device storage.

## Features
- Create trips with user-entered names
- Display trips in a dynamic SwiftUI list
- Swipe-to-delete interaction
- Persistent local storage using SwiftData
